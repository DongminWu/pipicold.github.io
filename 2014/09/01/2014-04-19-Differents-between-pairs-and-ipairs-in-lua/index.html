Lua语言中pairs和ipairs的区别

tbl = {“alpha”, “beta”, [“one”] = “uno”, [“two”] = “dos”}

for key, value in ipairs(tbl) do

        print(key, value)

end

-pairs()函数基本和ipairs()函数用法相同， 区别在于：

pairs()可以遍历整个table，即包括数组及非数组部分。

–>如有pairs迭代输出如下：

–>1 alpha

–>2 beta

–>one uno

–>two dos

ipairs()函数用于遍历table中的数组部分。

–>如有ipairs迭代输出如下：

–>1 alpha

–>2 beta
